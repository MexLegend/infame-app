## Cloning the repository

```shell
git clone https://github.com/MexLegend/infame-app.git
```

## Getting Started

Install the dependencies:

```sh
$ pnpm i
# or
$ npm i
```

Setup .environment file

```js
URI: 
CLOUDINARY_NAME:
CLOUDINARY_PRESET:
```

Start the app

```sh
$ pnpm start
# or
$ npm run start
```
